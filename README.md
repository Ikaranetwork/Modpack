# Overview
1.21.4 Modpack for Ikara Network designed to enhance gameplay and simulate 1.8 PvP while providing the best optimizations for the most fps and lowest input latency!

# Requirements
- Fabric 1.21.4
- Java 21

- Sodium
- Legacy Blocking

- Lithium
